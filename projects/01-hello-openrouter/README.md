# 01 - Hello OpenRouter

A simple hello world project to test OpenRouter with the free Grok model.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

## What it does

- Connects to OpenRouter API
- Sends a simple prompt to `x-ai/grok-4.1-fast:free`
- Prints the response and token usage

## Model

Using [x-ai/grok-4.1-fast:free](https://openrouter.ai/x-ai/grok-4.1-fast:free) - a free tier Grok model from xAI.
